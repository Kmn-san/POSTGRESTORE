# POSTGRESTORE

## About
A project who able user to post items.

## How It Works

### Fearures
- **User Authentication**: Secure user registration and login system with password stored as salted hashes.
- **Item Management**: Create, read, update and delete items
- **Item Listing**: Browse all available items in an organized manner

### Technology Stack
- **Backend** : Node.js with Express.js
- **Database** : PostgreSQL
- **Frontend** : HTML, JavaScript, React
